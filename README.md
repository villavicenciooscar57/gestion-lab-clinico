# Sistema de Gestión para Laboratorio Clínico

Este proyecto es una aplicación web diseñada para la automatización de procesos en un laboratorio clínico, permitiendo el registro de pacientes, la asignación y reporte de exámenes, y el control de historial clínico.

## Tecnologías Utilizadas
- **Backend:** Go (Golang)
- **Base de Datos:** Supabase (PostgreSQL)
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)

## Funcionalidades
- **Registro de Pacientes:** Gestión centralizada de datos y exámenes solicitados.
- **Reporte de Resultados:** Interfaz ágil para la carga de datos clínicos.
- **Historial Clínico:** Consulta por cédula y generación de reportes imprimibles.

## Instalación
1. Clonar el repositorio: `git clone https://github.com/villavicenciooscar57/gestion-lab-clinico.git`
2. Configurar las variables de entorno para Supabase.
3. Ejecutar el servidor: `go run ./cmd/lab`
